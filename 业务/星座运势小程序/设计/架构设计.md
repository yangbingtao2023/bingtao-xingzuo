## 架构总览

```
┌─ 数据管道（Python，每日定时）──┐
│  fetch (RSS + TianAPI)        │
│  → transform (标准化)          │
│  → publish (Cloud Base Open API)
└──────────┬───────────────────┘
           ▼
┌─ 微信云开发（Cloud Base）─────┐
│  fortunes 集合（12条/天）       │
│  daily_meta 集合（单例）        │
└──────────┬───────────────────┘
           ▼ wx.cloud.callFunction()
┌─ 云函数（Node.js）────────────┐
│  getDailyFortune              │
│  getFortuneDetail             │
└──────────┬───────────────────┘
           ▼
┌─ 小程序前端（原生 + Vant）────┐
│  首页 3×4 星座网格             │
│  → 运势详情页                  │
│  → 广告位（流量主）             │
│  → 本地缓存（6h TTL）          │
└──────────────────────────────┘
```

## 技术选型

| 层 | 选型 | 理由 |
|----|------|------|
| 前端 | 原生 + Vant Weapp | 单平台，无跨端开销 |
| 后端 | 微信云开发 | 免 ICP 域名，免费配额 |
| 内容 API | TianAPI（天聚数行）+ RSS 兜底 | 100 次/天免费，中文内容 |
| 数据管道 | Python 3.12 + uv | 复用现有 fetcher |
| 定时 | GitHub Actions | 免费，免维护 |

## 缓存策略

| 层 | 机制 | TTL |
|----|------|-----|
| 云开发数据库 | Python 管道每日推送 | 永久（按 date 覆盖） |
| 小程序本地 | wx.setStorageSync | 6 小时 |
